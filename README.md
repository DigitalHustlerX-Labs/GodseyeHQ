## Quick Start (Local Dev)

```bash
# Clone
git clone https://github.com/digitalhustlerx/Godseye.git
cd Godseye

# Install
npm install

# Configure
cp .env.example .env.local
# Add your GEMINI_API_KEY to .env.local

# Run
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

---

## Build & Deploy

```bash
npm run build        # Produces dist/ (Vite + esbuild)
npm start            # Run production server
```

Behind nginx:

```nginx
location / {
    proxy_pass http://127.0.0.1:3000;
    proxy_set_header Host $host;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    proxy_set_header X-Forwarded-Proto $scheme;
}
```

## API

### POST `/api/playground/generate`

Simulates how GodsEye handles a command. Powered by Gemini.

```json
{
  "command": "Create a new blog post about AI trends"
}
```

**Response:** `telegramResponse` (string) + `wordpressAction` (type, title, content, status)

Action types: `CREATE_POST`, `ACTIVATE_PLUGIN`, `DEACTIVATE_PLUGIN`, `WOOCOMMERCE_ORDER`, `SITE_HEALTH`, `ELEMENTOR_EDIT`, `MEDIA_UPLOAD`, `UNKNOWN`
